## 🌐Sitio Web

- Crear tu propio sitio web y cómprate un dominio.
- Siempre compra por separado el nombre de tu dominio del hospedaje del sitio web.
- Puedes comenzar con un simple archivo HTML con tus datos y una foto...
- O puedes usar alguna [plantilla gratis](https://colorlib.com/wp/templates/) que se adapte a tus necesidades...
- O puedes usar [WordPress.org](https://wordpress.org/).

## 😍Comunidad

Herramientas para comenzar a crear comunidad:

- **Lista de Correos**: [TinyLetter](https://tinyletter.com/)
- **Formularios**: [FormSubmit](https://formsubmit.co/)
- **Compartir en Redes Sociales**: [AddThis](https://www.addthis.com/)
- **Chats en vivo**:
  - [Drift](https://www.drift.com/)
  - [Canales de Slack](https://slack.com/intl/es-mx/)
  - Grupos de WhatsApp
- **Encuestas**: [Google Forms](https://docs.google.com/forms/)
- **Comentarios**: [Disqus](https://disqus.com/)
- **Analíticas**: [Google Analytics](https://analytics.google.com/)
- **Pagos en Línea**: [Paypal](https://www.paypal.com/)
